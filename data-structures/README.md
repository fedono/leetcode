- 之前一直不理解有人把数组的`push`,`pop`这些方法写出来，觉得有什么必要吗？是因为那时候不是很明白数据结构，数据结构的出现是为了要解决问题的，`list`，`tree`这些概念的产出，是为了解决特定的问题的，发展到如今，只有这几种数据结构，针对这几种数据结构有很多的场景去适用
